4. Teba chcem milovať ako učeníci, - krvou lásku stvrdiť ako mučeníci. - [:K tebe vzdychám, teba vzývam. - Svätý, svätý, svätý! spievam. - Ó, Sviatosť prevelebná, - večne buď pochválená!:]
